>基于Vue+Element实现的手机购物网站。本仓库为用户端网站仓库，管理端仓库在隔壁[Halo-MS](https://github.com/tozlam/Halo-MS)

>项目预览地址：[http://tozlam.cn:8888](http://tozlam.cn:8888)

#### Build Setup
````
安装依赖
npm install

开发模式
npm run dev

打包
npm run build
````
