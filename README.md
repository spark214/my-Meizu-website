>基于Vue+Element实现的Halo.手机购物网站。本仓库为用户端网站仓库，管理端仓库：[Halo-MS](https://github.com/tozlam/Halo-MS)。后台仓库：[Halo手机商城服务端](https://github.com/MelloChan/halo-server)。

>项目预览地址：[http://tozlam.cn:8888](http://tozlam.cn:8888)

>项目持续更新中。

#### Build Setup
````
安装依赖
npm install

开发模式
npm run dev

打包
npm run build
````

## 技术选型

- 框架：Vue 2.5.11
- UI库：Element 2.3.8
- HTTP连接工具：axios
- 脚手架工具 vue-cli
- 模块打包器：Webpack 3.6.0
- 图片懒加载工具：vue-lazyload
- 路由管理器：Vue Router
- 部署环境：CentOS 7.2 64位
- Web服务器: Nginx 1.14.0
- 版本控制: Git
- 开发工具: IDEA 2018
- 开发环境: Windows 10
